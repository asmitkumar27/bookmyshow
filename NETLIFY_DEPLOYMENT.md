# 🚀 Netlify Deployment Guide

## Step 1: Netlify Account Banao

1. **https://netlify.com** par jao
2. **"Sign up"** button click karo
3. **"Sign up with GitHub"** select karo
4. GitHub se authorize karo

---

## Step 2: GitHub Repository Connect Karo

1. Netlify dashboard mein **"Add new site"** → **"Import an existing project"** click karo
2. **GitHub** select karo
3. Authorize Netlify ko GitHub access dene ke liye
4. Apni repository **"asmitkumar27/bookmyshow"** select karo

---

## Step 3: Build Settings

**Configure your build settings:**

- **Base directory**: (leave empty)
- **Build command**: (leave empty)
- **Publish directory**: `.` (current folder)

Click **"Deploy"** 🎉

---

## Step 4: Wait for Deployment

- Netlify automatically build karega
- 1-2 minit mein **"Published"** status dikhega ✅
- Aapka live link mil jayega! 🌐

---

## Your Live Link Will Be:

```
https://bookmyshow-asmit.netlify.app
```

Or Netlify automatically unique name dega!
